1) Creating a local repository and create a README

	git init Prueba
	cd Prueba
	gedit README

2) Adding to the repository files that are on the folder

	git add smaple.c

3) Commiting changes 

	git commit -m "some_message"

4) Creating a new repository online *Use the same name I guess...)


5) Connecting the two repositories
	
	git remote add origin https://github.com/user_name/Prueba.git

6) Pushing what is the local repository to the online repository

	git push -u origin master	

7) Downloading what is inside a repository
	
	git clone https://github.com/nidiacristina/Prueba.git

///////////////////////////////////////////// Working //////////////////////////////

8) After modifying, commiting the changes

	git commit -m "some_message"

9) Listing the commits in reverse cronological order. THe most recent commits show up first

	git log
   
   To show only the last two entries
	
	git log -p -2

   Specifying a format, time and in reference to an specific function
	
	git log --pretty=format:"%h - %an, %ar : %s"
	git log --since=2.weeks
	git log -S function_name

10) Pushing 

	git push origin master	

11) Reseting to that specified commit
